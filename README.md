# BulkWeb.io Dashboard

> A modern, full-featured web application for creating and managing websites individually or in bulk with an intuitive dashboard interface.

## 🚀 Live Demo

**[View Live Application](https://bulkweb-io-dashboard-chi.vercel.app/)**

## 📋 Project Overview

BulkWeb.io Dashboard is a comprehensive frontend application built as a technical challenge project. It provides users with a complete platform for website creation and management, featuring template selection, domain management, hosting options, and bulk operations for efficient workflow.

This project demonstrates proficiency in modern React development, state management, TypeScript, and building scalable, user-friendly interfaces.

## ✨ Key Features

- **🏠 User Dashboard**: Centralized control panel for managing all website operations
- **🎨 Template Gallery**: Browse and select from a variety of pre-designed website templates
- **⚡ Bulk Operations**: Create and manage multiple websites simultaneously
- **🌐 Domain Management**: Integrated domain registration and management system
- **☁️ Hosting Options**: Multiple hosting solutions including self-hosted and managed options
- **📊 History Tracking**: Comprehensive history of all operations and changes
- **👤 User Authentication**: Secure login and profile management
- **💰 Pricing Plans**: Flexible pricing options for different user needs
- **🌍 Internationalization**: Multi-language support (i18n)
- **📱 Responsive Design**: Fully responsive interface for all device sizes
- **🗺️ Google Maps Integration**: Location-based features and mapping

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/kristipapallazo/bulkweb.io-dashboard.git
```

2. **Navigate to the project directory**
```bash
cd bulkweb.io-dashboard
```

3. **Install dependencies**
```bash
npm install
```

4. **Start the development server**
```bash
npm run dev
```

The application will open at `http://localhost:5173` (or the next available port).

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint for code quality checks

## 🌐 Deployment

This project is deployed on Vercel. To deploy your own instance:

1. Fork or clone this repository
2. Import the project to Vercel
3. Configure environment variables (if needed)
4. Deploy!
